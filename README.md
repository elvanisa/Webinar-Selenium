# Webinar-Selenium
Selenium Python - Webinar 5 May 2023

HOW TO RUN THE SCRIPT

1. Install Python
2. Clone Repository git@github.com:elvanisa/Webinar-Selenium.git
3. Open cmd or terminal
4. Go to the project path cd {your-path}/Webinar-Selenium
5. python3 -m venv env
6. activate the env
    source env/bin/activate # in Linux/Mac
    .\env\Scripts\activate # in Windows
7. pip install selenium
8. pip install HTMLTestRunner-rv
9. python3 runner.py # run the suite
